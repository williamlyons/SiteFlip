Lab 1: Priority Queue

Due: Due: 24 February 2016 at 2:30 PM

Abstract: We have studied the Binary Heap data structure in class. Now it is your turn to implement one. In support of both sorting and the next programming assignment, we will implement a minimum priority queue, so that elements are extracted from smallest to largest.

Goals of this lab:

1. Understand how a binary tree is implemented in an array.

2. Understand the details of implementing a binary heap.

Your Work for This Lab

  • Implement the methods as described in the videos and in the comments present in the files you are given.
  • Run the JUnit test TestMinHeap as you work to make sure you are on track.
  • When you have everything working, run the HeapTimer experiment, which will produce CSV data in the outputs folder.
  • Plot the time taken for your implementation. It should look like an n log n plot.
  • Save the plot in your outputs folder.

How to Submit Your Work

  • Make sure all of your work is commited in your repository.
  • Your work will be graded as of the due date for this assignment.
